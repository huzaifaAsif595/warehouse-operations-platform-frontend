🖥️ Warehouse Management System (WMS) – Frontend
🚀 Overview

This project represents the frontend of an enterprise Warehouse Management System (WMS), built to manage inventory, warehouse operations, and logistics workflows.

It provides a modular UI for handling real-time warehouse activities such as stock management, order processing, and system configuration.

⚠️ This is a sanitized version of a production system. Sensitive logic and integrations have been removed.

🧩 Core Features
📦 Inventory Management
Inventory master and mapping
Location assignment
Stock tracking and adjustments
🔄 Warehouse Operations
Move items between locations
Manual transactions
System replenishment
📊 Order & Transaction Handling
Order allocation and de-allocation
Transaction processing
Batch and cycle count management
🖨️ Configuration & Utilities
Printer setup
Admin preferences
Dialogs and shared components
📈 Reporting
Warehouse reports and data visualization
🛠️ Tech Stack
Angular
TypeScript
SCSS
REST API integration
